## 一、标题标签（Heading）
用于定义不同层级的标题，语义化标签，h1一个页面建议仅用一次。

| 标签   | 英文全称  | 示例代码                            |
| ------ | --------- | ----------------------------------- |
| `<h1>` | Heading 1 | `<h1>一级标题（只能使用一次）</h1>` |
| `<h2>` | Heading 2 | `<h2>二级标题</h2>`                 |
| `<h3>` | Heading 3 | `<h3>三级标题</h3>`                 |
| `<h4>` | Heading 4 | `<h4>四级标题</h4>`                 |
| `<h5>` | Heading 5 | `<h5>五级标题</h5>`                 |
| `<h6>` | Heading 6 | `<h6>六级标题</h6>`                 |

## 二、文本基础格式标签
| 标签       | 英文全称        | 功能说明                 | 示例代码                    |
| ---------- | --------------- | ------------------------ | --------------------------- |
| `<p>`      | Paragraph       | 定义段落                 | `<p>段落内容</p>`           |
| `<br>`     | Break Line      | 强制换行（单标签）       | `<p>第一行<br>第二行</p>`   |
| `<hr>`     | Horizontal Rule | 水平线（单标签）         | `<hr>`                      |
| `<b>`      | Bold            | 粗体（仅样式）           | `<b>粗体文本</b>`           |
| `<strong>` | Strong Emphasis | 粗体（语义化，强调重要） | `<strong>重要粗体</strong>` |
| `<i>`      | Italic          | 斜体（仅样式）           | `<i>斜体文本</i>`           |
| `<em>`     | Emphasized      | 斜体（语义化，强调）     | `<em>强调斜体</em>`         |
| `<u>`      | Underline       | 下划线（仅样式）         | `<u>下划线文本</u>`         |
| `<ins>`    | Inserted Text   | 下划线（语义化，插入）   | `<ins>插入的文本</ins>`     |
| `<s>`      | Strikethrough   | 删除线（仅样式）         | `<s>删除线文本</s>`         |
| `<del>`    | Deleted Text    | 删除线（语义化，删除）   | `<del>删除的文本</del>`     |

> 补充：HTML标签优先选**语义化标签**（如`<strong>`/`<em>`/`<ins>`/`<del>`），而非仅样式标签（`<b>`/`<i>`/`<u>`/`<s>`），提升网页可访问性。

## 三、媒体标签

| 标签      | 英文全称 | 功能说明           | 示例代码                                                     |
| --------- | -------- | ------------------ | ------------------------------------------------------------ |
| `<img>`   | Image    | 插入图片（单标签） | `<img src="assets/image/avatar.jpg" alt="头像">`             |
| `<audio>` | Audio    | 插入音频           | `<audio src="assets/audio/最终幻想.mp3" controls></audio>`   |
| `<video>` | Video    | 插入视频           | `<video src="assets/video/有法.mp4" controls width="400"></video>` |

> 补充：`controls` 属性用于显示音频/视频的播放控件，`alt` 是图片的替代文本（图片加载失败时显示，提升可访问性）。

## 四、布局与通用标签
| 标签     | 英文全称 | 显示类型 | 功能说明               | 示例代码                               |
| -------- | -------- | -------- | ---------------------- | -------------------------------------- |
| `<div>`  | Division | 块级元素 | 通用容器，用于布局分区 | `<div>块级内容</div>`                  |
| `<span>` | Span     | 行内元素 | 通用容器，包裹行内文本 | `<p>普通文本<span>高亮文本</span></p>` |

## 五、列表标签
### 1. 无序列表
| 标签   | 英文全称       | 功能说明     | 示例代码                                |
| ------ | -------------- | ------------ | --------------------------------------- |
| `<ul>` | Unordered List | 无序列表容器 | `<ul><li>条目1</li><li>条目2</li></ul>` |
| `<li>` | List Item      | 列表条目     | 嵌套在`<ul>/<ol>/<dl>`中使用            |

### 2. 有序列表
| 标签   | 英文全称     | 功能说明     | 示例代码                                |
| ------ | ------------ | ------------ | --------------------------------------- |
| `<ol>` | Ordered List | 有序列表容器 | `<ol><li>条目1</li><li>条目2</li></ol>` |

### 3. 定义列表
| 标签   | 英文全称               | 功能说明         | 示例代码       |
| ------ | ---------------------- | ---------------- | -------------- |
| `<dl>` | Definition List        | 定义列表容器     | 见下方完整示例 |
| `<dt>` | Definition Term        | 定义术语（标题） |                |
| `<dd>` | Definition Description | 定义描述（内容） |                |

**定义列表示例**：
```html
<dl>
    <dt>子标题1</dt>
    <dd>条目1</dd>
    <dd>条目2</dd>
    <dt>子标题2</dt>
    <dd>条目1</dd>
    <dd>条目2</dd>
</dl>
```

## 六、表格标签
| 标签/属性   | 英文全称          | 功能说明           |
| ----------- | ----------------- | ------------------ |
| `<table>`   | Table             | 表格容器           |
| `<caption>` | Caption           | 表格标题（语义化） |
| `<tr>`      | Table Row         | 表格行             |
| `<th>`      | Table Header Cell | 表头单元格         |
| `<td>`      | Table Data Cell   | 数据单元格         |
| `colspan`   | Column Span       | 跨列合并（横向）   |
| `rowspan`   | Row Span          | 跨行合并（纵向）   |

**表格完整示例**：
```html
<table border="1" cellspacing="0">
    <caption>表格标题</caption>
    <tr>
        <th>列1</th>
        <th colspan="2">列2（跨列合并）</th>
    </tr>
    <tr>
        <td>行1</td>
        <td rowspan="2">行1.5（跨行合并）</td>
        <td>行1</td>
    </tr>
    <tr>
        <td>行2</td>
        <td>行2</td>
    </tr>
</table>
```

## 七、表单相关标签
| 标签         | 英文全称  | 功能说明         | 示例代码                                            |
| ------------ | --------- | ---------------- | --------------------------------------------------- |
| `<select>`   | Select    | 下拉菜单容器     | 见下方完整示例                                      |
| `<option>`   | Option    | 下拉菜单选项     | 嵌套在`<select>`中使用                              |
| `<input>`    | Input     | 输入框（单标签） | `<input type="radio" name="gender" value="male">男` |

```html
<select>
    <option>选项1</option>
    <option>选项2</option>
    <option>选项3</option>
</select>
```
