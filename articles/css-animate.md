# 🔹平面转换
- **核心属性**：`transform`
- **作用**：为元素添加动态效果，通常与`transition`过渡配合使用
- **概念**：改变盒子在平面内的形态，包含**位移、旋转、缩放、倾斜**四种效果



## 📌基础转换



### ✨平移

- **核心属性**：`transform: translate(X轴距离, Y轴距离);`

- **单独设置轴**：`translateX(距离)` / `translateY(距离)`

- **取值规则**：
  1. 像素单位数值（px）
  2. 百分比（**参照盒子自身尺寸**计算）
  3. 正负均可（正：右/下；负：左/上）
  
  

### ✨旋转
- **核心属性**：`transform: rotate(旋转角度);`
- **角度单位**：deg（度数）
- **取值规则**：正负均可（正：顺时针旋转；负：逆时针旋转）
- **默认旋转原点**：盒子的**中心点**



#### 🔶改变转换原点

- **核心属性**：`transform-origin: 水平原点位置 垂直原点位置;`

- **取值规则**：

  1. 方位名词（left/top/right/bottom/center）
  2. 像素单位数值（px）
  3. 百分比（参照盒子自身尺寸）

  

### ✨缩放

- **核心属性**：
  1. 等比例缩放：`transform: scale(缩放倍数);`
  2. 单独设置轴：`transform: scale(X轴倍数, Y轴倍数);`
- **取值规则**：
  1. 倍数无单位（纯数字）
  2. 大于1：放大元素；小于1：缩小元素
- **优势**：缩放不会改变元素的实际占位，不会影响其他布局



### ✨倾斜

- **核心属性**：`transform: skew(倾斜角度);`
- **角度单位**：deg（度数）
- **取值**：正负均可，分别对应不同方向的倾斜



## 多重转换

- **书写技巧**：**先平移，再旋转**（`transform: translate() rotate();`）
- **核心原理**：旋转会改变元素的坐标轴方向，若先旋转，后续平移会沿旋转后的坐标轴生效，导致效果偏离预期
- **本质**：以第一种转换的坐标轴为基准进行形态改变



## 📌渐变

- **作用**：为盒子背景添加**多个颜色逐渐过渡**的效果，替代纯色/图片背景
- **核心属性**：`background-image`（渐变本质是背景图）
- **分类**：线性渐变、径向渐变



### ✨线性渐变
- **语法**：
  ```css
  background-image: linear-gradient(
    渐变方向, /* 可选，默认从上到下 */
    颜色1 终点位置, /* 可选，百分比/px */
    颜色2 终点位置,
    ...
  );
  ```
- **渐变方向取值**：
  1. 方位名词：`to left`/`to right`/`to top`/`to bottom`/`to top left`（对角）
  2. 角度度数：`30deg`/`180deg`（0deg为从下到上，度数增加顺时针旋转）
- **终点位置**：可选，百分比表示颜色过渡的结束位置，不写则均匀过渡



### ✨径向渐变
- **作用**：从**中心点**向四周发散的渐变效果，常用作按钮高光
- **语法**：
  ```css
  background-image: radial-gradient(
    半径 at 圆心位置, /* 必写，半径可设2个值为椭圆 */
    颜色1 终点位置,
    颜色2 终点位置,
    ...
  );
  ```
- **取值规则**：
  1. 半径：像素/百分比，2个值表示椭圆（如`50px 30px`）
  2. 圆心位置：方位名词/像素/百分比（如`center center`/`50px 50px`）
  3. 终点位置：同线性渐变，可选
  
  

# 🔹空间转换
- **核心属性**：`transform`
- **空间定义**：由X、Y、Z三条坐标轴构成立体空间，**Z轴与视线方向一致**（指向观察者）
- **视觉效果**：近大远小、近实远虚



## 📌空间平移
- **语法**：
  1. 复合写法：`transform: translate3d(x, y, z);`
  2. 单独设置轴：`translateX(距离)`/`translateY(距离)`/`translateZ(距离)`
- **取值**：像素/百分比，正负均可（Z轴正：向观察者靠近；负：远离）
- **注意**：默认情况下Z轴平移无视觉效果，需配合**视距perspective**使用



### ✨视距 perspective

- **作用**：指定观察者与`Z=0`平面的距离，为元素添加3D透视效果
- **属性特点**：**添加给元素的父级**，子级会继承透视效果
- **推荐取值**：800px - 1200px（取值越小，透视效果越强）
- **语法**：`perspective: 1000px;`



## 📌空间旋转

- **基础语法**：`transform: rotateX(deg)`/`rotateY(deg)`/`rotateZ(deg)`
  - rotateZ：与2D旋转一致，绕Z轴顺时针/逆时针旋转
  - rotateX/Y：绕X/Y轴的立体旋转
- **旋转方向判断**：**左手法则**
  1. 左手握住旋转轴，拇指指向坐标轴**正值方向**
  2. 其余四指的弯曲方向，即为旋转的**正值方向**
- **自定义旋转轴**：`transform: rotate3d(x, y, z, 角度);`
  - x/y/z：取值0-1之间的数字，表示旋转轴在该方向的分量
  - 例：`rotate3d(1,1,0,45deg)` 绕X+Y轴的对角线旋转45度



## 📌立体呈现 transform-style

- **作用**：设置元素的子元素是处于3D空间还是平面中，实现真正的3D效果
- **核心属性**：`transform-style: preserve-3d;`（关键值，需添加给父级）
- **取值**：
  1. `preserve-3d`：子级处于**3D空间**（实现立体必备）
  2. `flat`：默认值，子级处于**平面中**，无3D效果
- **3D立体图形实现步骤**：
  1. 父元素添加`transform-style: preserve-3d;` + 透视`perspective`
  2. 子级元素设置**定位**（脱离标准流，避免重叠）
  3. 调整子盒子的位置（位移/旋转），形成3D形态
- **提示**：3D空间内，每个转换元素都有独立的坐标轴，互不干扰



## 📌空间缩放

- **语法**：
  1. 复合写法：`transform: scale3d(x, y, z);`
  2. 单独设置轴：`scaleX(倍数)`/`scaleY(倍数)`/`scaleZ(倍数)`
- **取值**：纯数字，大于1放大，小于1缩小
- **作用**：沿X/Y/Z轴单独缩放，实现3D的拉伸/压缩效果



# 🔹动画（animation）



## 📌动画与过渡的区别

| 特性     | 过渡（transition）              | 动画（animation）                |
| -------- | ------------------------------- | -------------------------------- |
| 状态数   | 仅支持**两个状态**（初始/结束） | 支持**多个状态**自由定义         |
| 可控性   | 触发后自动执行，无额外控制      | 可控制重复播放、延迟、暂停、方向 |
| 执行方式 | 需鼠标悬停/点击等**触发条件**   | 可自动执行，无需触发             |



## 📌动画实现



### ✨定义动画（@keyframes 关键字，两种写法）

```css
/* 写法1：从初始状态到结束状态（from→to） */
@keyframes 动画名称 {
  from { 初始样式; }
  to { 结束样式; }
}

/* 写法2：多状态自定义（百分比，0%→100%） */
@keyframes 动画名称 {
  0% { 初始样式; }
  50% { 中间样式; }
  100% { 结束样式; }
}
```


### ✨使用动画（给元素添加animation属性）

```css
.box {
  animation: 动画名称 动画时长; /* 必写两个值：名称+时长 */
}
```



## 📌复合属性

- **语法**：

```css
animation: 名称 时长 [速度曲线] [延迟时间] [重复次数] [动画方向] [执行后状态];
```
- **属性**：

| 属性名                    | 作用         | 核心取值                                         |
| ------------------------- | ------------ | ------------------------------------------------ |
| animation-name            | 绑定动画名称 | 与@keyframes定义的名称一致                       |
| animation-duration        | 动画时长     | 秒（s）/毫秒（ms），必写                         |
| animation-timing-function | 速度曲线     | linear（匀速）/steps(数字)（逐帧）               |
| animation-delay           | 延迟执行时间 | s/ms，默认0                                      |
| animation-iteration-count | 重复次数     | 数字/infinite（无限循环）                        |
| animation-direction       | 动画执行方向 | alternate（反向循环）                            |
| animation-fill-mode       | 执行后状态   | forwards（保持最后一帧）/backwards（回到第一帧） |
| animation-play-state      | 暂停/播放    | paused（暂停）/running（播放），常配合:hover使用 |


## 📌多组动画

- **作用**：为一个元素同时添加多个独立的动画效果
- **语法**：多个动画用**逗号**分隔，各自的属性独立生效
```css
.box {
  /* 例：逐帧精灵动画 + 匀速移动动画 */
  animation: run 1s steps(12) infinite, move 3s linear forwards;
}
```
