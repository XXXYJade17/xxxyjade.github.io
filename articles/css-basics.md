# 🔹CSS 导入方式



## 📌内部编写

在HTML的 `<head>` 标签内，通过 `<style>` 标签直接编写CSS代码，适合单页面少量样式、快速调试的场景。

```HTML
<head>
 	<style>
 		body { 
 			background-color: #f0f0f0; /* 设置页面背景色为浅灰色 */
 		}
 	</style>
</head>
```



## 📌外部导入

在HTML的 `<head>` 标签内，通 `<link>` 标签引入外部 `.css` 文件，是企业开发中最常用的方式，适合多页面共享样式的场景。

```html
<head>
 	<link rel="stylesheet" href="style.css">
</head>
```



# 🔹CSS 选择器



## 📌基础选择器



### ✨标签选择器

通过 **HTML 标签名**选中页面中所有该类型的元素，批量设置统一样式。

- 语法格式：

```css
lable_name {
 	property: value;
}
```

- 示例：选中所有 `<div>` 标签，设置文字大小为14px

```CSS
div {
	font-size: 14px;
}
```



### ✨类选择器

通过自定义的 `class` 属性选中元素，支持“多元素共享同一样式”，是实际开发中最灵活、最常用的选择器。

- 语法格式：

```CSS
.class_name {
	property: value;
}
```

- 示例：定义 `content` 类，为所有绑定该类的元素设置颜色

```CSS
.content {
 	color: #333;
}
```

```HTML
<p class="content">第一段文本</p>
```



### ✨ID选择器

通过元素的**唯一** `id` 属性选中指定元素，一个HTML页面中 `id` 名称不可重复，适用于为单个元素设置专属样式。

- 语法格式：

```CSS
#id_name {
	property: value;
}
```

- 示例：选中 `id="header"` 的元素，设置背景色

```CSS
#header {
  	background-color: #fff;
}
```

```HTML
 <header id="header">页面头部</header>
```



### ✨通配符选择器

用 `*` 匹配页面中**所有HTML元素**，常用于初始化页面默认样式（如清除浏览器默认的内外边距）。

- 语法格式：

```CSS
* {
  	属性: 值;
}
```

- 示例：清除所有元素的默认内外边距

```CSS
* {
  	margin: 0;
  	padding: 0;
}
```



## 📌特殊选择器



### ✨复合选择器

| 选择器类型 | 语法格式                              | 核心说明                                                 |
| ---------- | ------------------------------------- | -------------------------------------------------------- |
| 后代选择器 | `父选择器 后代选择器 { 属性: 值; }`   | 选中父选择器下**所有层级**的后代元素，空格分隔           |
| 子代选择器 | `父选择器 > 子代选择器 { 属性: 值; }` | 仅选中父选择器下**直接子元素**（最近一级子代），`>` 分隔 |
| 并集选择器 | `选择器1, 选择器2, ... { 属性: 值; }` | 多组选择器共享相同样式，逗号分隔（可跨类型）             |
| 交集选择器 | `选择器1选择器2 { 属性: 值; }`        | 选中**同时满足所有选择器条件**的元素，无分隔符           |



### ✨伪类选择器



#### 🔶超链接相关伪类（元素状态类）

| 伪类     | 语法                           | 说明                               |
| -------- | ------------------------------ | ---------------------------------- |
| :link    | `选择器:link { 属性: 值; }`    | 选中未被访问过的超链接             |
| :visited | `选择器:visited { 属性: 值; }` | 选中已被访问过的超链接             |
| :hover   | `选择器:hover { 属性: 值; }`   | 选中鼠标悬停的元素（不限于超链接） |
| :active  | `选择器:active { 属性: 值; }`  | 选中元素被点击激活的瞬间状态       |

> **建议按 LVHA 顺序书写**（:link → :visited → :hover → :active）



#### 🔶结构伪类选择器（元素位置类）

| 伪类          | 语法                                     | 说明                                                         |
| ------------- | ---------------------------------------- | ------------------------------------------------------------ |
| :first-child  | `E:first-child { 属性: 值; }`            | 查找父元素下**第一个**E类型子元素                            |
| :last-child   | `E:last-child { 属性: 值; }`             | 查找父元素下**最后一个**E类型子元素                          |
| :nth-child(n) | `E:nth-child(n/含n表达式) { 属性: 值; }` | 查找父元素下第n个E类型子元素；n可填数字/关键字（even/odd）/表达式（如2n、n+3） |



### ✨伪元素选择器

| 伪元素   | 语法                                   | 核心说明                                                     |
| -------- | -------------------------------------- | ------------------------------------------------------------ |
| ::before | `E::before { content: ""; 属性: 值; }` | 在E元素内部最前面插入虚拟元素；必须设置`content`（可留空），默认行内显示 |
| ::after  | `E::after { content: ""; 属性: 值; }`  | 在E元素内部最后面插入虚拟元素；必须设置`content`（可留空），默认行内显示 |

> 伪类 vs 伪元素 核心区别：
>
> ​	伪类：针对「元素已有状态/位置」做样式修改，无新元素生成
>
> ​	伪元素：创建「虚拟元素」承载装饰样式，DOM中无真实节点，需配合`content`属性



# 🔹CSS 三大特性



## 📌继承性

- 子级元素会**默认继承父级元素的文字控制类属性**，无需重复编写，实现代码复用。

- 示例代码

```css
/* 父级div设置文字样式 */
.father {
  color: #666;
  font-size: 14px;
  line-height: 1.5;
}
```

```html
<div class="father">
  <p>子级p标签会继承父级的颜色、字号、行高</p>
  <a href="#">子级a标签不会继承颜色（默认蓝色），但会继承字号、行高</a>
  <h5>子级h5标签不会继承字号（默认更大），但会继承颜色、行高</h5>
</div>
```



## 📌层叠性

- 层叠性是CSS处理**同一元素被多个样式规则匹配**的基础规则，仅适用于**选择器类型相同**的场景。

- 核心规则：
  1. **相同属性覆盖**：如果多个样式给同一元素设置了**相同的CSS属性**，**后面编写的样式**会覆盖前面的样式；
  2. **不同属性叠加**：如果多个样式给同一元素设置了**不同的CSS属性**，则所有属性都会生效，样式叠加。


- 示例代码：

```css
/* 同一选择器，多次设置样式 */
.box {
  color: red; /* 相同属性，会被后面的green覆盖 */
  font-weight: 700; /* 不同属性，会叠加生效 */
}
.box {
  color: green; /* 覆盖前面的red */
  font-size: 20px; /* 不同属性，叠加生效 */
}
```

```html
<div class="box">文字为绿色、20px、加粗</div>
```



## 📌优先级

- **不同类型选择器**给同一元素设置样式时的冲突问题，**优先级高的样式会生效**。

- 选中元素的范围**越大**，优先级**越低**；手动设置的样式优先级高于继承的样式。



### ✨基础选择器的优先级顺序

- 从低到高依次排列，层级差距不可跨越（作用范围越小，优先级越高）

```
通配符选择器 < 标签选择器 < 类选择器 < ID选择器 < 行内样式 < !important
```

- 示例代码

```css
* { color: black !important; } /* 最高，最终生效 */
div { color: red; } /* 标签选择器 */
.box { color: green; } /* 类选择器 */
#box { color: blue; } /* ID选择器 */
```

```html
<div id="box" class="box" style="color: purple;">文字最终为黑色</div>
```



### ✨复合选择器的权重叠加计算

- 叠加公式：用四元组表示，**从左到右依次比较**，每一级之间**不存在进位**。

```
(行内样式个数, ID选择器个数, 类选择器个数, 标签选择器个数)
```

- 计算规则
  1. 先比较第一个数字（行内样式数），个数多的优先级高；
  2. 若第一个数字相同，再比较第二个（ID数），以此类推；
  3. 若四元组完全相同，遵循**层叠性**，后面的样式覆盖前面的。


- 叠加计算示例

| 复合选择器          | 权重四元组 | 优先级排序 |
| ------------------- | ---------- | ---------- |
| `div span`          | (0,0,0,2)  | 最低       |
| `.box span`         | (0,0,1,1)  | 中等       |
| `.box #content`     | (0,1,1,0)  | 较高       |
| `div .box #content` | (0,1,1,1)  | 最高       |

> 特殊规则：
>
> 1. `!important` 写在属性值后，会**忽略所有权重计算**，直接成为最高优先级；
> 2. **继承的样式权重为0**，无论父级选择器优先级多高，子级继承的样式都不如子级自身的任意选择器样式。



# 🔹盒子模型（从内到外）

所有HTML元素都是一个**矩形盒子**，盒子模型定义了元素的尺寸、间距和边框的计算规则，是CSS布局的基础。



## 📌内容区（Content）

- 作用：存放元素的文字、图片等实际内容

- 控制属性：`width`（宽度）、`height`（高度），单位常用`px`

- 示例：`width: 200px; height: 150px;`



## 📌内边距（Padding）

- 作用：**内容区与边框之间的空隙**，仅在盒子内部生效，会撑大盒子

- 单方向设置：`padding-top`/`padding-right`/`padding-bottom`/`padding-left`

- 简写规则（**顺时针，缺值取对面**）：

  | 取值个数 | 语法                            | 含义                     |
  | -------- | ------------------------------- | ------------------------ |
  | 1个      | `padding: 10px;`                | 上下左右内边距都是10px   |
  | 2个      | `padding: 10px 20px;`           | 上下10px，左右20px       |
  | 3个      | `padding: 10px 20px 30px;`      | 上10px，左右20px，下30px |
  | 4个      | `padding: 10px 20px 30px 40px;` | 上→右→下→左 依次对应     |

  

## 📌边框（Border）

- 作用：包裹盒子的线条，分隔内边距和外边距

- 复合写法（推荐）：`border: 粗细 样式 颜色;`（顺序可换，缺一不可）

- 线条样式：`solid`（实线，最常用）、`dashed`（虚线）、`dotted`（点线）

- 示例：`border: 2px solid #333;`

- 单方向边框：`border-top: 1px red solid;`（仅设置上边框）



## 📌外边距（Margin）

- 作用：**盒子与其他盒子之间的间距**，在盒子外部生效，不撑大盒子

- 写法：和`padding`完全一致（单方向+四值简写，顺时针规则）

- 实用技巧：**版心居中**（给固定宽度的盒子设置`margin: 0 auto;`，仅左右居中生效）

- 示例：`width: 1200px; margin: 0 auto;`（网页版心标准写法）
